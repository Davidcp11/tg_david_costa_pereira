# sample_data/cap3

Minimal data needed to regenerate the Cap3 figures without the full KITTI dataset.

| File | Origin | Used by |
|------|--------|---------|
| `rgb.png` | KITTI Odometry seq 00, frame 2000 (`image_2/002000.png`) | gen_depth_densification_en.py, gen_bbox_mask_en.py |
| `depth_sparse.png` | KITTI Depth Completion, seq 2011_10_03_drive_0027, frame 2000 | gen_depth_densification_en.py |
| `depth_dense.npy` | Densified depth from `depth_2/002000.npy` (generated by the morphological pipeline) | gen_depth_densification_en.py, gen_bbox_mask_en.py |

## Regenerate figures

```bash
# densification_kernels.pdf + densification_steps.pdf (no data needed — purely algorithmic)
python scripts/plot_densification_example.py --lang en
python scripts/plot_densification_example.py --lang pt

# depth_densification_example.pdf
python scripts/gen_depth_densification_en.py --lang en
python scripts/gen_depth_densification_en.py --lang pt

# bbox_mask_example.pdf
python scripts/gen_bbox_mask_en.py --lang en
python scripts/gen_bbox_mask_en.py --lang pt
```
